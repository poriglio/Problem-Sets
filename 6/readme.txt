node secondgreatlow.js <comma-separated list of numbers>

This program will return first the 
second-lowest and then the second-highest 
of the numbers. Do not put spaces between the
number in the list, only commas!

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

node timeconvert.js <number>

This program will return the number of hours
and the number of minutes the number converts to,
separated by a colon.

-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=

node brackets.js <string>

This program will determine whether the brackets are
correctly matched in your string.
It considers {} and ().
