Problem Set 5

  • alphabetSoup(arg) - This function expects a string with only letters, no punctuation or numbers, and returns the string with the characters in alphabetical order.
  • vowelCount(arg)- This function expects a string, and returns the number of vowels in the string.
  • coinDeterminer(arg)- This function expects an integer ranging from 1 to 250, and returns the smallest number of coins you can use to add up to that integer. The coin values are 11, 9, 7, 5, and 1.
