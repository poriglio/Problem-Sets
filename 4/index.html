Problem Set 4

  • Exercise 1:
    1. Each string will containt at least one non-number.
    2. 55Hello and 5Hello5 should return different answers.
    3. Search for all numbers in a string.
    4. Add the numbers and return the sum.
  • Exercise 2:
    1. Return the longest word in a single string.
    2. If 2+ words are the same length, return the first one.
    3. Ignore punctuation and assume the input sentence will not be empty.
  • Bonus Exercise:
    1. Search for all numbers in a single string parameter.
    2. Add the numbers and divide that number by the number of letters in the string.
    3. Round to the nearest whole number.
    4. No digit will appear next to another digit, i.e. no "hello44444world"
    5. Each string will have at least one letter.
